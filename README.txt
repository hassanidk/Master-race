 Utilisation du framework PHASER, de coffeescript, gulp et nodejs

 // gulp est une alternative à grunt
 // Conseillé d'installer globalement
 sudo npm install gulp -g

 // Installation des dépendances
 npm install

 // Install du jeu
 npm run build

 // Faire tourner le serveur
 npm run serve

 // Build et serveur
 npm run all

 Pour modifier les modes (développement, release...) : 

  - Modifier le fichier app/coffee/config.env.coffee

  - avec les types définis dans le fichier app/coffee/config.environment-vars.coffee
